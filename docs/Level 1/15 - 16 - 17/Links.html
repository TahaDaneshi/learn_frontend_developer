'a' tag attributes table

Attribute Purpose SEO Importance
href Destination URL Essential
target How to open link (_blank for new tab) Medium
rel Link relationship (e.g., nofollow for sponsored links) High
title Additional description (hover text) Low
download For downloadable files Medium

جدوله attributes تگ a

اتریبیوت کاربرد اهمیت سئو
href آدرس مقصد ضروری
target نحوه بازکردن لینک (_blank برای تب جدید) متوسط
rel رابطه لینک (مثلاً nofollow برای لینک‌های تبلیغاتی) بالا
title توضیح اضافی (توصیف hover) کم
download برای فایل‌های قابل دانلود متوسط
